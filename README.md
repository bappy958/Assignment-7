# 👥 KeenKeeper — Keep Your Friendships Alive

## 📌 Project Description

KeenKeeper is a modern web application designed to help users maintain meaningful relationships. It tracks interactions, highlights overdue connections, and provides visual insights into communication habits through an analytics dashboard.

---

## 🚀 Technologies Used

* **Frontend:** Next.js 
* **Styling:** Tailwind CSS
* **Charts & Visualization:** Recharts
* **State Management:** React Context API
* **Deployment:** Vercel

---

## ✨ Key Features

### 1. 📊 Friendship Analytics Dashboard

Interactive pie chart visualization to track communication types like calls, texts, and video interactions.

### 2. ⏰ Smart Reminder System

Automatically identifies friends you haven’t contacted recently and marks them as overdue.

### 3. 🧑‍🤝‍🧑 Friend Management System

Add, organize, and manage friends with tags and personalized notes.
